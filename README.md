# 视频分割拼接工具 V2.5.4

一个功能强大的视频分割和拼接工具，支持自定义分割比例、封面设置、音频配置等高级功能。

**最新版本 V2.5.4** - 视频叠加预览修复 + 多实例目录同步！

## 功能特点

- ✅ **视频叠加** - 全帧叠加模式，前景视频居中叠加在背景视频上
- ✅ **灵活分割** - 支持左右/上下分割，自定义分割比例（10%-90%）
- ✅ **智能拼接** - 多种拼接模式（A+C、A+D、B+C、B+D、四宫格）
- ✅ **曲线分界线** - 支持 Catmull-Rom 样条曲线分界线，专业级曲线编辑器
- ✅ **封面设置** - 从视频帧或图片创建封面，支持实时预览
- ✅ **音频控制** - 支持模板/列表/混合/静音/自定义音频，独立音量调节
- ✅ **批量处理** - 一次处理多个视频，支持独立设置和一键同步
- ✅ **可视化预览** - 实时预览分割效果、拼接效果、高清放大预览
- ✅ **位置顺序** - 自由控制模板和列表视频的拼接顺序
- ✅ **透明通道** - 自动检测 MOV alpha 通道，ProRes 4444 编码输出
- ✅ **缩放模式** - 适应/填充/拉伸三种缩放模式，支持自定义缩放百分比
- ✅ **输出尺寸** - 跟随模板/跟随列表/自定义尺寸，内置多种预设
- ✅ **目录记忆** - 三个文件选择按钮各自独立记忆路径，多实例间热更新同步
- ✅ **智能错误诊断** - FFmpeg 错误自动分析，提供具体修复建议


## 使用说明

### Windows

1. 解压下载的 ZIP 文件
2. 双击运行 `VideoSplitTool.exe`
3. FFmpeg 已内置，无需额外安装

### macOS

1. 解压下载的 ZIP 文件
2. 安装 FFmpeg（如果还没有）：
   ```bash
   brew install ffmpeg
   ```
3. 运行程序：
   - 双击 `VideoSplitTool`
   - 如果提示无法打开，在终端执行：
     ```bash
     chmod +x VideoSplitTool
     ./VideoSplitTool
     ```

## 主要功能

### 1. 视频分割
- 左右分割或上下分割
- 自定义分割位置（10%-90%）
- 可视化拖拽调整分割线

### 2. 视频拼接
- 模板视频 + 列表视频
- 多种组合模式
- 自定义位置顺序（模板在前/列表在前）

### 3. 封面设置
- 从模板视频选帧
- 从列表视频选帧
- 导入外部图片
- 实时预览封面效果
- 自定义封面时长（0.1-10秒）

### 4. 音频配置
- 仅模板音频
- 仅列表音频
- 混合音频
- 静音
- 自定义音频文件

### 6. 曲线分界线
- Catmull-Rom 样条平滑曲线
- 专业级曲线编辑器（键盘微调、精确坐标输入）
- 多种预设曲线（S曲线、波浪、弧形等）
- 超采样抗锯齿 + 高斯模糊边缘处理

### 7. 缩放与输出
- 三种缩放模式：适应（留黑边）、填充（裁切）、拉伸
- 模板视频和列表视频可独立设置缩放模式
- 输出尺寸：跟随模板/跟随列表/自定义（内置竖屏/横屏/正方形预设）
- 可视化效果示意图，支持拖拽调整区块比例

### 8. 智能诊断
- FFmpeg 错误自动分析，识别 10+ 种常见错误
- 提供具体的修复建议
- 完整的输入验证（视频文件、分割比例、目录权限等）
- 添加多个视频
- 每个视频独立设置
- 一键应用设置到全部
- 同步模板分割比例

## 项目结构

```
VideoSplitTool/
├── main.py                          # 主入口（Tkinter 窗口 + 退出保护）
├── app.py                           # 备用入口
├── requirements.txt                 # 依赖（Pillow）
├── VideoSplitTool.spec              # PyInstaller 打包配置
├── logs/                            # 日志目录（按日期归档，自动清理）
├── dist/                            # 打包输出目录
├── build/                           # 构建临时目录
│
└── src/                             # 源代码（四层分层架构）
    ├── __init__.py                  # 版本信息
    │
    ├── models/                      # 数据模型层
    │   ├── config.py                # 全局配置（枚举 + dataclass）
    │   │   - SplitMode              #   分割模式（水平/垂直）
    │   │   - PositionOrder          #   位置顺序（模板优先/列表优先）
    │   │   - OutputSizeMode         #   输出尺寸模式
    │   │   - ScaleMode              #   缩放模式（适应/填充/拉伸）
    │   │   - AudioSource            #   音频来源
    │   │   - AppConfig              #   全局配置（支持序列化/反序列化）
    │   │   - MergeConfig            #   拼接配置（A/B/C/D 组合）
    │   │   - DialogDirsConfig       #   目录记忆配置
    │   └── video_item.py            # 视频列表项数据模型
    │       - VideoItem              #   封面设置、分割比例、曲线点等
    │
    ├── core/                        # 核心业务层
    │   ├── ffmpeg_utils.py          # FFmpeg/FFprobe 路径查找 + 视频信息获取
    │   │   - get_ffmpeg_path()      #   自动查找本地 FFmpeg
    │   │   - FFmpegHelper           #   视频信息、帧提取、图片转视频
    │   │   - VideoInfo              #   视频元信息（分辨率/时长/音频/透明通道）
    │   ├── video_processor.py       # 视频处理核心（~1274 行）
    │   │   - VideoProcessor         #   主处理器
    │   │     - process_videos()     #   统一入口
    │   │     - _build_filter_complex()       # 标准分割拼接滤镜链
    │   │     - _build_overlay_filter_complex()  # 视频叠加滤镜链
    │   │     - _build_alpha_filter_complex()     # 透明通道滤镜链
    │   │     - _build_mask_filter_complex()      # 曲线蒙版滤镜链
    │   │     - _build_horizontal_filter()        # 水平分割子滤镜
    │   │     - _build_vertical_filter()          # 垂直分割子滤镜
    │   │     - _add_cover_to_video()             # 封面添加
    │   └── error_handler.py         # 智能错误诊断 + 输入验证
    │       - ErrorDiagnostics       #   FFmpeg 错误模式匹配 + 修复建议
    │       - InputValidator         #   视频/目录/比例/时长验证
    │
    ├── ui/                          # UI 界面层（Mixin 组合架构）
    │   ├── main_window.py           # 主窗口 VideoSplitApp
    │   │   - 继承 6 个 Mixin，组合所有功能
    │   │   - 配置持久化到 ~/.video_split_tool/settings.json
    │   │   - 多实例热更新目录同步
    │   ├── mixins/                  # 功能 Mixin 模块
    │   │   ├── preview_mixin.py     #   预览渲染（模板帧、拼接模拟、高清放大预览）
    │   │   ├── processing_mixin.py  #   批量处理线程、进度回调、结果展示
    │   │   ├── audio_mixin.py       #   音频源选择、音量调节、ffplay 试听
    │   │   ├── cover_mixin.py       #   封面类型/帧时间/图片管理
    │   │   ├── divider_mixin.py     #   曲线分界线（Catmull-Rom 样条 + 超采样蒙版）
    │   │   └── diagram_mixin.py     #   效果示意图（可拖拽调整区块比例）
    │   ├── dialogs.py               # 对话框
    │   │   - VideoSettingsDialog    #   单个视频设置（分割比例调整）
    │   │   - CurveEditorDialog      #   曲线分界线编辑器（专业级，支持键盘微调）
    │   ├── widgets.py               # ScrollableFrame 可滚动容器
    │   └── compat.py                # 兼容性函数桥接

    └── utils/                       # 工具函数层
        ├── file_utils.py            # 文件路径、临时目录、视频格式校验
        ├── format_utils.py          # 时长/尺寸/方向格式化显示
        ├── logger.py                # 日志系统（按日期归档，自动清理 7 天）
        └── temp_manager.py          # 临时文件追踪与清理（自动清理 3 天）
```

## 架构设计

### 四层分层

```
┌─────────────────────────────────┐
│         UI 界面层 (ui/)          │  Tkinter GUI + Mixin 组合
├─────────────────────────────────┤
│        核心业务层 (core/)        │  FFmpeg 命令构建与执行
├─────────────────────────────────┤
│        数据模型层 (models/)      │  dataclass + Enum 定义
├─────────────────────────────────┤
│        工具函数层 (utils/)       │  日志、临时文件、格式化
└─────────────────────────────────┘
```

### 关键设计亮点

| 特性 | 说明 |
|------|------|
| **Mixin 组合** | 主窗口通过多继承组合 6 个功能 Mixin，降低耦合 |
| **智能错误诊断** | FFmpeg stderr 模式匹配，自动识别 10+ 种常见错误并给出修复建议 |
| **超采样抗锯齿** | 曲线蒙版先创建 2 倍大小图像再缩小 + 高斯模糊，获得平滑边缘 |
| **线程安全** | 处理线程通过 `root.after()` 调度 UI 更新，非守护线程保证处理完整性 |
| **预览防抖** | 滑块拖拽时轻量更新分割线 + 延迟完整刷新，保证流畅体验 |
| **目录热更新** | 多实例间通过配置文件热更新同步目录记忆 |
| **退出保护** | 处理中关闭窗口弹出确认，防止输出文件损坏 |
| **宽高偶数约束** | `_make_even()` 确保输出尺寸符合 libx264 编码要求 |
