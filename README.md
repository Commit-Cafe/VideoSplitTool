# 视频分割拼接工具 V2.3

一个功能强大的视频分割和拼接工具，支持自定义分割比例、封面设置、音频配置等高级功能。

**最新版本 V2.3** - 文件选择独立路径 & MOV透明通道支持！

## 功能特点

- ✅ **灵活分割** - 支持左右/上下分割，自定义分割比例
- ✅ **智能拼接** - 多种拼接模式（A+C、A+D、B+C、B+D、四宫格）
- ✅ **封面设置** - 从视频帧或图片创建封面，支持实时预览
- ✅ **音频控制** - 支持模板音频、列表音频、混合、静音或自定义音频
- ✅ **批量处理** - 一次处理多个视频，支持独立设置
- ✅ **可视化预览** - 实时预览分割效果和封面帧
- ✅ **位置顺序** - 自由控制模板和列表视频的拼接顺序
- ✅ **透明通道** - 支持MOV格式透明通道，自动检测alpha通道


## 使用说明

### Windows

1. 解压下载的 ZIP 文件
2. 双击运行 `VideoSplitTool.exe`
3. FFmpeg 已内置，无需额外安装

### macOS

1. 解压下载的 ZIP 文件
2. 安装 FFmpeg（如果还没有）：
   ```bash
   brew install ffmpeg
   ```
3. 运行程序：
   - 双击 `VideoSplitTool`
   - 如果提示无法打开，在终端执行：
     ```bash
     chmod +x VideoSplitTool
     ./VideoSplitTool
     ```

## 主要功能

### 1. 视频分割
- 左右分割或上下分割
- 自定义分割位置（10%-90%）
- 可视化拖拽调整分割线

### 2. 视频拼接
- 模板视频 + 列表视频
- 多种组合模式
- 自定义位置顺序（模板在前/列表在前）

### 3. 封面设置
- 从模板视频选帧
- 从列表视频选帧
- 导入外部图片
- 实时预览封面效果
- 自定义封面时长（0.1-10秒）

### 4. 音频配置
- 仅模板音频
- 仅列表音频
- 混合音频
- 静音
- 自定义音频文件

### 5. 批量处理
- 添加多个视频
- 每个视频独立设置
- 一键应用设置到全部
- 同步模板分割比例

## 系统要求

- **Windows**: Windows 10 及以上
- **macOS**: macOS 10.13 及以上
- **依赖**: FFmpeg（macOS 需手动安装）

## 技术栈

- Python 3.11+
- Tkinter (GUI)
- FFmpeg (视频处理)
- Pillow (图像处理)
