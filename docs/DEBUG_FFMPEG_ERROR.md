# FFmpeg 参数错误诊断指南

## 常见错误原因

### 1. 音频配置错误

**错误场景**：
- 选择"模板音频"但模板视频没有音频
- 选择"列表音频"但列表视频没有音频
- 选择"混合音频"但两个视频都没有音频

**错误信息**：
```
Stream map '[outa]' matches no streams
```

**解决方法**：
- 使用"静音"选项
- 或添加自定义音频
- 或检查视频是否有音频轨道

---

### 2. 自定义音频文件问题

**错误场景**：
- 音频文件路径包含中文或特殊字符
- 音频文件格式不支持
- 音频文件不存在或已删除

**错误信息**：
```
自定义音频文件不存在
No such file or directory
```

**解决方法**：
- 使用英文路径
- 确保音频文件存在
- 使用常见格式（mp3, wav, aac）

---

### 3. 封面拼接音频丢失

**错误场景**：
- 设置了封面，但主视频有音频
- 封面时长设置过长

**错误信息**：
```
anullsrc filter error
concat filter error
```

**临时解决方法**：
- 暂时不使用封面功能
- 或选择"静音"选项

---

### 4. 视频格式问题

**错误场景**：
- 视频文件损坏
- 不支持的视频编码
- 视频尺寸异常（0x0）

**错误信息**：
```
Invalid data found
Video encoding failed
视频尺寸无效
```

**解决方法**：
- 使用其他播放器测试视频是否能正常播放
- 使用 HandBrake 等工具重新编码视频
- 确保视频分辨率正常

---

## 快速诊断步骤

### Step 1: 查看详细错误信息

程序失败后，在结果窗口中查看：
```
状态: 失败
原因: [错误信息]
```

记下完整的错误信息。

### Step 2: 检查视频文件

1. **播放测试**
   - 模板视频能正常播放吗？
   - 列表视频能正常播放吗？
   - 音频能正常播放吗？

2. **音频检测**
   - 打开视频，查看是否有声音
   - 如果没有声音，不要选择"模板音频"或"列表音频"

### Step 3: 简化配置测试

从最简单的配置开始：

**测试 1: 基础拼接（无封面、静音）**
- 模板视频：选择一个
- 视频列表：添加一个
- 拼接设置：勾选 A 和 C
- **音频设置：选择"静音"**
- **封面设置：选择"无封面"**
- 点击"开始处理"

✅ 如果成功 → 逐步添加功能
❌ 如果失败 → 视频文件本身有问题

**测试 2: 添加音频**
- 在测试 1 基础上
- 音频设置：选择"模板音频"
- 处理

✅ 如果成功 → 音频功能正常
❌ 如果失败 → 音频配置有问题

**测试 3: 添加封面**
- 在测试 1 基础上
- 封面设置：选择"从视频选择帧"
- 帧来源：模板视频
- 时间点：1秒
- 封面时长：1秒
- 处理

✅ 如果成功 → 封面功能正常
❌ 如果失败 → 封面功能有问题

## 已知问题和解决方案

### 问题 1: 音频选择"模板音频"但视频无声音

**症状**:
```
错误: Stream map '[outa]' matches no streams
或
错误: matches no streams
```

**原因**: 代码尝试提取不存在的音频流

**解决方法**:
1. 选择"静音"
2. 或使用有声音的视频
3. 或选择"自定义音频"添加外部音频

### 问题 2: 封面拼接后没有声音

**症状**: 输出视频前几秒（封面部分）没有声音

**原因**: 这是正常的，封面部分本就是静音图片

**解决方法**:
- 这是预期行为，不是错误
- 如需封面有声音，可使用"自定义音频"功能

### 问题 3: 特殊字符路径导致失败

**症状**:
```
错误: No such file or directory
或
错误: 文件不存在
```

**原因**: 路径包含中文、空格或特殊字符

**解决方法**:
- 将视频文件移动到纯英文路径
- 例如: `D:\videos\` 而不是 `D:\我的视频\`
